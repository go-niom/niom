## Description

[Niom](https://github.com/niomgo/niom) framework starter repository.

## Installation

  ```bash 
  $ go install github.com/go-niom/niom@latest  

  #Get List commands 
  $ niom -h
  ```

## Spin up your server
 ```bash
 $ niom new appname 
 ```
## Running the app
  ```bash
# development mode
$ niom start:dev

# spin up
$ niom start

```
